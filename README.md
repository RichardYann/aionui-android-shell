# AionUi Shell for Android

Lightweight Android shell for AionUi WebUI.  
一个面向 AionUi WebUI 的轻量 Android 客户端。

Bring your AionUi workspace to Android with a focused shell built for quick access, server switching, immersive navigation, and practical daily use.  
将你的 AionUi 工作空间带到 Android 设备上，用更轻量、更顺手的方式完成快速连接、服务器切换、沉浸浏览与日常使用。

## ✨ Features / 功能

- Connect to any AionUi WebUI server with a simple URL-based setup
- Save multiple servers, rename them, mark favorites, and reopen recent ones quickly
- Switch servers directly inside the app without leaving the current workspace flow
- Automatically reopen the last active server on later launches
- Navigate with back, refresh, and change-server controls
- Support file upload through standard web file inputs
- Download files through Android system DownloadManager
- 首次启动可通过 URL 快速连接任意 AionUi WebUI 服务
- 支持保存多个服务器，并可自定义命名、重命名、收藏和快速访问最近使用项
- 支持在应用内直接切换服务器，不打断当前使用节奏
- 后续启动可自动重新打开最近一次使用的服务器
- 提供返回、刷新、切换服务器等常用导航能力
- 支持网页中的标准文件上传能力
- 支持通过 Android 系统下载管理器下载文件

## 🌊 Highlights / 特点

- Lightweight and focused: open AionUi fast without extra native complexity
- Server friendly: manage personal, lab, and team endpoints in one place
- Immersive navigation: the top bar stays hidden until you intentionally reveal it
- Clear loading feedback: connection and refresh states are easier to understand
- AionUI branding: launcher icon and app presence better reflect the product identity
- 轻量专注：无需复杂原生层，就能快速进入 AionUi 工作界面
- 多服务器友好：个人、实验室、团队环境都能集中管理
- 沉浸体验：顶部导航默认隐藏，需要时再唤出
- 加载清晰：连接与刷新状态更容易理解
- 品牌一致：图标与整体观感更贴近 AionUI 产品形象
